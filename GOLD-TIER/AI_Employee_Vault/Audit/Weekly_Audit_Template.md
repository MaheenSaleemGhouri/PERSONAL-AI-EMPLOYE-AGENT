# Weekly Audit Template

## Instructions for Claude
Every Sunday at 8:00 PM, follow these steps exactly:

1. Read Business_Goals.md — note monthly targets and alert thresholds
2. Read all files in /Done/ created this week — count and categorize completed tasks
3. Read Accounting/Current_Month.md — note revenue, expenses, flagged transactions
4. Read all files in /Social/Posted/ from this week — count posts per platform
5. Read /Logs/ for this week — count errors, notable actions
6. Check /Pending_Approval/ — note anything overdue (>24 hours)
7. Generate CEO Briefing and save to /Briefings/YYYY-MM-DD_Monday_Briefing.md
8. Update Dashboard.md Gold section with latest numbers
9. Create task file in /Needs_Action/ for any flagged items needing attention

## CEO Briefing Must Include
- Executive summary (3 sentences max)
- Revenue vs target
- Completed tasks this week
- Bottlenecks (tasks that took too long or failed)
- Social media performance per platform
- Subscription audit flags
- Proactive suggestions (max 3)
- Upcoming deadlines next 7 days
